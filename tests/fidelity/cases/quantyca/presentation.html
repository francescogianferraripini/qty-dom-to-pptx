01 / Introduzione

PERCHÉ REVEAL.JS

Slide come codice. Niente build, zero vendor lock-in.

HTML & CSS

Un file HTML, un file CSS, font locali. Apri e proietta — nessun tooling richiesto.

Versionabile

Vive su Git come il resto del codice. Diff leggibili, review via PR.

Estendibile

Plugin per code highlight, chart, math, markdown, speaker view.

Export PDF

Stampa pixel-perfect con ?print-pdf o decktape.

Deep-link

Ogni slide ha un hash URL. Condividi il link e atterri esattamente lì.

Accessibile

Testo vero, screen-reader friendly, prefers-reduced-motion onorato.
